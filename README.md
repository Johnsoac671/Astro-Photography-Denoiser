# Astro-Photography-Denoiser
Comparing different methods for denoising astrophotography images to aid in galaxy classification

## Datasets:
K. W. Willett et al., “Galaxy Zoo 2: detailed morphological classifications for 304,122 galaxies from the Sloan Digital Sky Survey,” 2013, doi: 10.48550/ARXIV.1308.3496. (Galaxy Zoo 2)

R. E. Hart et al., “Galaxy Zoo: comparing the demographics of spiral arm number and a new method for correcting redshift bias,” Mon. Not. R. Astron. Soc., vol. 461, no. 4, pp. 3663–3682, Oct. 2016, doi: 10.1093/mnras/stw1588. (Galaxy Zoo 2 - debiased data)
